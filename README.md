# Sobre o Projeto

Sangue e Poder é um RPG de mesa original inspirado no gênero de Super-Heróis Sombrios.

O cenário apresenta um mundo moderno onde indivíduos com habilidades extraordinárias surgiram através de:

- Mutações Genéticas
- Engenharia Biológica
- Experimentos Científicos
- Programas Militares Secretos
- Tecnologia Avançada
- Evolução Artificial

Neste universo, super-heróis não são necessariamente heróis.

Fama, dinheiro, influência política, corrupção e manipulação da mídia são tão importantes quanto superforça ou visão de calor.

O sistema utiliza mecânicas d20 inspiradas em RPGs modernos, adaptadas para um cenário realista, brutal e cinematográfico.

# Principais Características

## Sistema d20

O jogo utiliza:

- d20
- Classe de Armadura
- Pontos de Vida
- Iniciativa
- Vantagem
- Desvantagem
- Testes de Resistência

---

## Sistema de Poderes

A magia foi completamente removida.

No lugar dela existem:

- Poderes
- Espaços de Poder
- Sobrecarga
- Despertar Evolutivo
- Formas Evolutivas
- Poderes Definitivos

Os poderes são classificados do Nível 1 ao Nível 9.

---

## Sistema de Reputação

Os personagens possuem:

- Popularidade
- Temor
- Credibilidade

Suas ações influenciam diretamente a forma como o mundo os enxerga.

---

## Sistema de Corrupção

Todo personagem pode cair moralmente.

A corrupção é medida por pontos acumulados durante a campanha.

Os níveis vão de:

- Estável
- Influenciado
- Corrompido
- Degradado
- Monstruoso
- Ruína Moral

---

## Sistema de Sanidade

A exposição constante à violência pode causar:

- Estresse
- Traumas
- Fobias
- Dependências
- Colapsos Mentais

# Filosofia do Sistema

Sangue e Poder não é um jogo sobre vencer.

É um jogo sobre escolhas.

A pergunta central do cenário é:

"O que alguém faz quando possui poder suficiente para mudar o mundo?"
